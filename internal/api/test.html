Bounce Test Console

v1.2 — 12 endpoints | FPB + TugaBasket

GET/api/competitions

Lista de competicoes conhecidas (hardcoded)



  
GET/api/game/{internalID}
Ficha de jogo com periodos e estatisticas. Ex: 413420


  
GET/api/games?club=ID
Jogos de um clube. Ex: clube 119 (Benfica)


  
GET/api/games?competition=ID&page=resultados
Jogos de uma competicao (calendario ou resultados)


  
GET/api/standings/{compID}
Classificacao de uma competicao (fases via AJAX)


  
GET/api/athlete/{id}
Perfil de atleta: nome, foto, stats por epoca, biografia


  
GET/api/team/{id}
Detalhe de equipa: plantel com fotos + jogos. Ex: equipa_59060


  
GET/api/club/{clubID}/teams
Lista de equipas de um clube. Ex: clube 127


  
GET/api/elo
Ranking ELO nacional


  
GET/api/predictions/{gameID}
Previsao de vencedor baseada em ELO






  
GET/api/tugabasket/standings?competitionId=ID
Classificacoes do TugaBasket (accordion HTML)
